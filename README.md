# 4pet

The mobile application for 4PET - The social network for your pet.
<p align="center">
  <img src="./docs/images/welcome-img.png" width="317" height="449" style="text-align:center"/>
</b>

## Quick start

- Get packages from **pub** for Flutter app:

```bash
flutter pub get
```

- Run project:

```bash
flutter run
```
